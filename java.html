Java Lets is a coding competition organized in the java domain under Renaissance 2k18.
A technical event involving concepts of Core JAVA only.

GUIDELINES 1. Java Lets is a coding competition which will be organized on hackerearth so no one will be allowed copy the previously written code from anywhere, if someone founds guilty, the team will be disqualified.
2. Solutions for the problems will be given online.
3. Results will be given by the hackerearth on the basis of complexities and time of submission for the solutions of all the submission.

Round 1:
● It will be a MCQ round consisting of 30 questions in total.
● This round will be on Hackerearth.
● Contestants will be awarded 3 marks for each correct answer, -1 for each incorrect answer and 0 for unattempted questions.
● Top 30 teams from round 1 will be selected for round 2.
● Results will be generated by hackerearth.

Round 2:
● It will be coding round consisting of 3 questions.
● It will be organized online, on hackerearth.
● Evaluation will be done by hackerearth on the basis of code efficiency and time of submission.
